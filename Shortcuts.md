Shortcuts
=========

### Tools
- `Q` - Color picker
- `W` - Pencil
- `E` - Eraser

### File I/O
- `Command-S` - Save file
- `Command-O` - Open file

### Zoom
- `1` - 100% Zoom
- `2` - 200% Zoom
- `3` - 400% Zoom
- `4` - 800% Zoom
- `5` - 1600% Zoom

### Undo/Redo
- `Command-Z` - Undo change
- `Command-Y` - Redo change
- `Command-Shift-Z` - Redo change
